# Spotify-Inspired Dark Green Theme

## Color Palette

### Primary Colors
- **Spotify Green**: `#1DB954` - Primary brand color
- **Spotify Green Light**: `#1ED760` - Secondary/hover state
- **Background Dark**: `#121212` - Main background
- **Surface**: `#181818` - Card/component background
- **Surface Light**: `#282828` - Hover/elevated surfaces
- **Text White**: `#FFFFFF` - Primary text color

### Usage in Tailwind

```css
bg-spotify-dark       /* #121212 - Main background */
bg-spotify-surface    /* #181818 - Cards, surfaces */
bg-spotify-green      /* #1DB954 - Primary accent */
text-spotify-green    /* #1DB954 - Green text */
border-spotify-green  /* #1DB954 - Green borders */
hover:bg-spotify-green-light  /* #1ED760 - Hover state */
```

## Design System

1. **Navbar**
   - Glass effect with Spotify surface background

2. **Hero**
   - Spotify green animated music icon

   - Green primary CTA with black text
   - Surface cards with green borders on hover

3. **Features**
   - Black/transparent section background

4. **Stats**
   - Surface background section
   - Green icon backgrounds
   - Green text for statistics

5. **Commands**
   - Tab selector with green active state

   - Green icons and accents

6. **Installation**
   - Surface background section
   - Green step numbers and icons

7. **Footer**
   - Black background

8. **Particles Background**
